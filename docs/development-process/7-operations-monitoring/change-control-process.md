- Changes after Version 1.0.0 release
- Does not apply to bug fixes
- Does not apply to typo fixes

### 6. Implement change

### 7. [Inspect change](../5-construction/inspections/code-inspection.md)

### 8. Remove abstraction layer

### 9. Release change into production
